Other

Hide tags: Hide all tags from SCSS output.
Always hide: Space separated tag list that will always be hidden from output.
Never hide: Space separated tag list that will never be hidden from output.
Hide IDs: Hide all IDs from SCSS output.
Always hide: Space separated ID list that will always be hidden from output.
Do not include #.
Never hide: Space separated ID list that will never be hidden from output.
Do not include #.
Remove by Ids: Space separated ID list of which the selector and it's children will be removed from the SCSS heirarchy.
Do not include #.
Extract by Ids: Space separated ID list of which the selector and it's children will be extracted from their position in the SCSS heirarchy, and moved to the top level of the heirarchy. This is useful for componenets.
Do not include #.
Hide classes: Hide all classes from SCSS output.
Always hide: Space separated class list that will always be hidden from output.
Do not include '.'
Never hide: Space separated class list that will never be hidden from output.
Do not include '.'
Remove by class: Space separated class list of which the selector and it's children will be removed from the SCSS heirarchy.
Do not include '.'
Extract by class: Space separated class list of which the selector and it's children will be extracted from their position in the SCSS heirarchy, and moved to the top level of the heirarchy. This is useful for componenets.
Do not include '.'
Reduce Siblings: Identical sibling elements with identical children will only be output as a single selector. Most useful for repeating elements like list items.
Combine parents: A bottom up merger of similar parent elements. If two sibling selectors have parents with the same selector, then those parents will be merged.
Convert BEM: Recognizes and converts double _ BEM heirarchy to &__.
Does not currently identify --modifiers
